Urban Forest: Theme Design for Sandvox

Sandvox drew inspiration from the CSS Zen Garden and created one set of templates composed of the same markup structure and just "skinned" it with CSS. This is the Urban Forest theme I designed for them. This design has proven to be the most enduring design I have done. The design is still used in production by several websites (including this one for example).

The Design

This design was tailored to fit monitor resolutions of 800x640 or greater. The layout is an example of the "ice" method where the page is fixed-width, but always centered in the viewport. The theme supported layouts with one or two columns.

I modified some of my own photography to create the imagery in the masthead and the "watermark" image in the content area. I created "folliage" that tied the masthead to the content. A custom font for the header was applied using the Scalable Inman Flash Replacement implementation.
